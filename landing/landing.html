Quickstart

Up and Running in 60 Seconds

1

Install the SDK

pip install openinterest-lens

2

Get your API key

Sign up at openinterestlens.com — free tier includes ES, NQ, CL positioning signals.

3

First API call

Set OIL_API_KEY=oil_sk_live_... then call client.signals.positioning("ES")

4

Stream live updates

Connect to wss://api.openinterestlens.com/ws/v1/signals for real-time WebSocket feeds.

5

Go build

Integrate signals into your trading models, dashboards, or alerts. Check the SDK guide for advanced usage.

# 5 lines to your first signal
from openinterest_lens import OILClient

client = OILClient(api_key="oil_sk_live_...")
signal = client.signals.positioning("ES")
print(f"Smart money: {signal.smart_money.direction} (z={signal.smart_money.z_score})")
# → Smart money: bullish (z=2.34)
